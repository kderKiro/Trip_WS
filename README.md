# Trip_WS

A web platform that connects travel agencies and customers, allowing agencies to publish trip packages and customers to browse, register, and manage complete travel experiences including flights, accommodation, and related services.

## Live Demo

https://full-trip.onrender.com

## Features

* Agency account management
* Trip publication and management
* Customer registration and booking
* Flight and accommodation information
* User authentication
* Web-based administration dashboard

## Technologies Used

* React
* Flask
* Python
* PostgreSQL/MySQL
* Google Cloud Storage
